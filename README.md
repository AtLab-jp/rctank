RCtank
======

RC Tank Controlled by Raspberry Pi and Smartphone

## Requirement
- Raspbian
- Python 3.5.2
- WebIOPi


## Parts
- Raspberry Pi 3 Model B
- Motor driver(TA7291P) * 2
- [Twin motor gearbox](https://www.amazon.co.jp/gp/product/B001Q13BIU/)


## Installation
```sh
$ git clone https://github.com/AtLab-jp/rctank
$ cd rctank
```
